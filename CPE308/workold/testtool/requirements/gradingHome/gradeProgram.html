2.5.8 Grading Programming Question

This is the professor's home screen when trying to grade a Programming question.

  • Auto Grading: The program will either compare the contents of two different files or use the output of the provided script in order to generate a score.
  • Manual Grading: After evaulating the question, the student's answer, and the script output the professor can leave a note for the student and edit the point-value field by clicking on the grey box in order to input a score.
