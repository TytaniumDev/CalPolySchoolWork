TestTool

1.1 Problem Statement

The general problems to be addressed by the TestTool project are the following:
  • to reduce the need for paper in testing
  • to provide a fully customizable database for test questions
  • to provide an intuitive piece of software that allows teachers to easily generate tests for students
  • to provide a simple and helpful environment for students who are taking tests
  • to improve upon the blackboard test generation system
Although these more general problems have been addressed by many different and varying products that are available, there are specific areas in which TestTool improves upon and provides new features over these. The problems that these features attempt to fix are:
  • varying degrees of customization of all aspects to questions in the database
  • the ability to use questions that haven't been used since a certain time period ago
  • allow students to be able to utilize practice tests
TestTool also addresses its customer's specific requirements of features that aren't available in any current tool available, such as:
  • a focus on programming questions that can be automatically graded through the use of scripts
  • the ability to draw graphical objects to answer certain types of questions
  • the option to grade tests by each individual test or by individual question
