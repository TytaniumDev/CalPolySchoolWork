1.5.4 Exam-Software

"Exam Software is a Test Management Software to create and conduct computer based online examination. -from www.exam-software.com"

This software is very flexible and has a large variety of options in the way of question customization. Its major drawbacks are its lack of database support for "question banks" and doesn't have any programming specific features. Its user interface is a bit outdated, but it does an adequate job nonetheless. Here's a breakdown of its features:

Good Features:
  • can attach video, sound, and graphics to questions
  • can set window to always be on top, preventing distractions
Bad Features:
  • no programming specific features
  • antiquated user interface
Missing features:
  • database functionality doesn't exist
  • no simple way to re-use questions in different tests
