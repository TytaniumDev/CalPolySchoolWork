1.5.1 Adit Testdesk

The Adit Testdesk offers many features that a test creator will utilize. It is a highly customizable program that provides many tools for manual test generation. Testdesk's look, feel, and prestige suggest it is regarded as a well-made piece of software. However, it does not offer any options for automatic test-generation.

Good Features:
  • complete customization with a multitude of question types (including multiple choice, essay, gap fillused in a web browser, and thus is cross platform
  • ability to use plain text, tables, images, videos, embedded objects, and Adobe Flash as questions
  • comfortable UI that is very similar to Microsoft Office 2007
  • built in ability to test more complex questions with a script
  • topic support to organize tests based on question topics
  • preview question to make sure layout is correct
Bad Features:
  • customization may take a little while to get used to since there are many features that can be used
  • feedback for test-takers is not specific to each question, giving only a summary at the end
Missing Features:
  • ability to use questions from a database
  • no support for Mac OS X
  • auto-generation of basic tests is missing
