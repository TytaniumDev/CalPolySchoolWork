1.5.2 ExamBuilder

ExamBuilder is used in industry for training purposes for such companies as Wal-Mart, Microsoft, 3M, and ADP(among many others). It supports several “non-writing” question types, such as multiple choice(one or multiple answers), true/false, fill in the blank, and matching. ExamBuilder has a question pool that chooses questions based on subject-matter and students can be organized into different groups(i.e class/section). Student gets immediate feedback with results after submitting, in which instructors can manually add their personal feedback to.

Overall, ExamBuilder provides a decent level of functionality that runs in a web browser, but it does not support programming and short-answer/essay questions.

Good Features:
  • used in a web browser, and thus is cross platform
  • question pool, that allows instructor to categorize questions into similar topics
  • randomization, instructor sets the number of each type from each question pool
  • simple and intuitive interface
  • same website for both students and instructor
  • individual randomization of question order to each student
  • export of reports into PDF or Excel files
  • ExamBuilder will customize products to certain companies demands
Bad Features:
  • built more for industry(training) uses, rather than education
  • requires internet at all times you are using it
  • pricey for an individual to purchase
  • can only be run using their host, cannot be put onto a local(intranet) server
Missing Features:
  • lack of essay/short answer support
  • lack of coding environment
  • lack of consentual sharing between multiple instructors
