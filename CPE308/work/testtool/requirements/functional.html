2. Functional Requirements

The primary functionality of the Test Tools Suite is to generate tests to the teachers specification, and provide sufficient grading/reporting tools for test feedback. The Test Tools suite allows for a variety of question types, including coding questions with the ability to import a script for grading. It also maintains a database of test questions to be used for future tests or by other teachers of the same class.

Following an overview of the Test Tools User Interface, details of system use are presented in the following scenarios:
  • modifications to the questions database
  • a user generating a test in three different generation modes
  • a view of the proctor and student screen in a test taking environment
  • a view of the test reports generated after the students have completed the test
  • a view of the graded tests including breakdown by question
These scenarios cover all functional requirements for the system.
