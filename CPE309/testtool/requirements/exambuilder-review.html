1.5.4 Exam Builder Review

The Exam Builder is a commercially available test generating software focused on providing managers and instructions a quick, easy way to create and deliver tests. This is best illustrated through their design motto “Simplicity is Powerful.” Exam Builder has been around since 1998; its longevity acting as proof of the useful and easy-to-use nature of the software.

Good Features:
  • Questions on each generated test are placed in random order and multiple choice answers are scrambled to prevent cheating.
  • After each test is completed, each student can view how he scored on questions in each question category and thus can focus on improving weaknesses.
  • Questions are organized in question pools, which enables related questions to be grouped together by class, book, keyword, etc.
  • Timed tests are implemented smoothly
  • The use of graphs and charts in test questions is supported.
Bad Features:
  • There is no flexibility in the nature of test taking: they can only be taken online.
  • Each exam is graded in its entirety at once; the option to grade tests a question at a time does not exist.
  • Questions do not appear to be shared between tests.
Missing Features:
  • Each question does not contain a difficulty level, thus making it challenging to rank each test difficulty and reducing flexibility for the user.
  • No option for program code questions exist.
  • No option for essay questions exist.
